------

Keyphrase is a modern PHP library for generating secure passwords, passphrases, and BIP39 mnemonics with an immutable, fluent API.

## Requirements

> **Requires [PHP 8.4+](https://php.net/releases/)**

## Installation

```bash
composer require cline/keyphrase
```

- **[Passphrase Generation](DOCS.md#doc-docs-passphrase-generation)** - EFF diceware wordlists and options
